# v1.2.0 Release Notes

## Unified Candidate Ledger

All imports (DOCX, LinkedIn, GitHub) now merge into a **single candidate document** instead of creating separate documents. Source tracking on every element for full provenance.

- `ResumeDocumentMerger.MergeInto()` — deduplicates experience, skills, education, projects by matching on company/name + date overlap
- `ImportSources` on WorkExperience, Skill, Education, Project — tracks which imports contributed each element
- Cross-source anomaly detection: name mismatches, title differences, date discrepancies >3 months

## Looking / Hiring Toggle

Jobs page now has a **Looking / Hiring** mode toggle:

- **Looking**: existing candidate job search and matching
- **Hiring**: employer mode — structured role entry with **mandatory salary range and location**. No "competitive salary" allowed.
- `IsEmployerRole` flag separates candidate jobs from employer roles
- `EmployerProfile` model for company info

## Skill Community Visualization

UMAP dimensionality reduction + Leiden community characterisation:

- `SkillGraphProjection.ProjectTo2D()` — reduces 384-dim MiniLM embeddings to 2D scatter positions via UMAP
- `CharacteriseCommunities()` — auto-labels each Leiden community by its discriminative skills (not just frequency — internal-weight ratio + document frequency)
- ScottPlot scatter chart: skills as colored dots, position from UMAP, color from Leiden community
- Community labels below chart (e.g. "Cloud & DevOps (Docker, K8s, Terraform)")

## Dashboard Improvements

- **Skills Dashboard**: colorful category pie chart (LiveCharts2) with proper category distribution
- **Career Timeline**: proper Gantt chart (ScottPlot) with date-positioned horizontal bars per role, company labels, dynamic height
- **Skill Communities**: UMAP scatter plot with Leiden-colored clusters

## ScottPlot + LiveCharts2

Dual charting library setup:
- ScottPlot.Avalonia 5.1.58 for Gantt (horizontal bars + date axis) and scatter (UMAP communities)
- LiveChartsCore.SkiaSharpView.Avalonia 2.0.0 for pie chart and polar radar

Avalonia bumped to 11.3.4 across all projects.

## Career Planner Page

New "Career" page with gap analysis against a selected job:
- Quick Wins (low effort), Side Projects (medium), New Learning (high)
- Fit/coverage/actions-to-good-fit badges

## Other Changes

- PDF export via QuestPDF — professional formatting, cross-platform
- Leiden community detection (refinement phase over Louvain)
- Resume extraction RRF fusion (multi-source confidence boost)
- Temporal skill drift analysis across resume variants
- Experience deduplication with variant storage for user review
- Mobile project heads (Android + iOS) — scaffolded, requires workloads
- UMAP NuGet package (curiosity-ai) for dimensionality reduction
