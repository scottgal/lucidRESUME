# v1.0.4 Release Notes

## Pre-converted Resume NER Model

The resume NER model (`yashpwr/resume-ner-bert-v2`) is now pre-exported to ONNX and hosted on HuggingFace at [scottgal/resume-ner-bert-v2-onnx](https://huggingface.co/scottgal/resume-ner-bert-v2-onnx).

- **No Python dependency** -- the app downloads the pre-converted ONNX model directly on first launch
- **No more warnings on Windows** about missing Python or `optimum`
- Removed the entire Python export pipeline (`ExportResumeNerModelAsync`, `FindPython`)
- Same download pattern as the general NER model -- simple, reliable

## Build & Release Improvements

- **Single-file publishing** -- Windows and Linux builds are true single-file executables
- **macOS native lib handling** -- native `.dylib` files (SkiaSharp, ONNX Runtime, sqlite-vec) are kept alongside the executable instead of self-extracting, avoiding Gatekeeper/notarization issues with temp dir execution
- **Platform-appropriate archives** -- Windows releases are `.zip`, macOS and Linux are `.tar.gz` (no more redundant double-archiving)

## NuGet Package Icon

- Added the `ml` monogram icon to the `Mostlylucid.Avalonia.UITesting` NuGet package

## README

- Rewritten Getting Started section to be user-focused: download, extract, run
- AI configuration simplified -- Ollama install or Profile page, no `dotnet user-secrets` commands
- "Build from Source" section for developers kept but de-emphasised
