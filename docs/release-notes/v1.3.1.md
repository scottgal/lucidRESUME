# v1.3.1 Release Notes

## DocLayNet Layout Detection

Real ONNX inference for document structure detection:

- **YOLOv10m DocLayNet** (58MB) detects 11 region types from rendered page images
- SkiaSharp preprocessing: resize to 640x640 with letterboxing, CHW normalisation
- Morph renders DOCX → PNG inline when no cached images exist
- **Structural hash** from detected regions — correctly differentiates layouts that the old font-based fingerprint matched as identical
- Lazy-loaded: model only downloaded on first use, not at startup
- Hosted at [scottgal/doclaynet-yolov10m-onnx](https://huggingface.co/scottgal/doclaynet-yolov10m-onnx)

## CLI Tailor Command

`lucidresume tailor --resume cv.docx --job "JD text" [--output tailored.md]`

Full pipeline: parse → quality score → compress → LLM tailor → re-score → report delta.
Batch tested on 4 resumes: consistent +5 to +8 point quality improvement.

## CLI Drift Command

`lucidresume drift --resume1 old.docx --resume2 new.docx`

Compares skill ledgers between two resume versions. Reports added, dropped, strengthened, and weakened skills.

## Light/Dark Theme Toggle

System/Light/Dark dropdown on the Profile page. Switches `Application.RequestedThemeVariant` at runtime. Light theme already defined in `AppTheme.axaml`.

## Fluent Icons

Navigation sidebar now uses FluentIcons (Document, DataBarVertical, Rocket, Briefcase, Add, Send, DataFunnel, Settings, QuestionCircle).

## Import Review Page

Built but wiring in progress. When importing a second source (LinkedIn on top of DOCX), shows a review page with accept/reject per item before merging.

## Bug Fixes

- **LLM skill recovery awaited**: all CLI commands now use `ParseHelper.ParseAndAwaitAsync` to wait for background LLM skill recovery. Fixed 0-skill parse on narrative resumes.
- **JD parser "Remote" as company**: "Senior .NET Developer - Remote" no longer treats "Remote" as the company name.
- **Ollama availability check**: CLI tailor command now triggers `CheckAvailabilityAsync` before checking `IsAvailable` flag.

## Template Collection

7 resume templates downloaded for parser testing (Chronological, Harvard, Modern, Two-Column, Technical, CV, Non-technical). Each gets a unique structural hash from DocLayNet.
